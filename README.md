# Saucedemo Playwright Automation Test

This Playwright test automates a complete user flow on the public demo site [Saucedemo](https://www.saucedemo.com):

1. Login with valid credentials  
2. Add the first product to cart  
3. Verify product name appears correctly in cart  
4. Logout and return to login page  

---

## Prerequisites

- Node.js (v16 or higher)  
- npm  
- Git

---

## How to Run

```bash
# 1. Clone the repo
git clone https://github.com/your-username/saucedemo1-playwright-test.git
bash# 2. Enter project directory
cd saucedemo1-playwright-test
bash# 3. Install dependencies
npm install
bash# 4. Run tests (headless by default)
npx playwright test
bash# 5. View with UI
npx playwright test --ui
